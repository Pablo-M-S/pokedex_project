# Pokédex System - C++

## About the Project

Pokédex System is a C++ application developed to organize and manage Pokémon data by region.

The project was created to practice and apply fundamental programming concepts, including data structures, vectors, structs, functions, and file handling.

Currently, the application stores Pokémon information such as ID and name, and saves the data into text files for persistence.

## Features

- Pokémon data organization by region
- Storage using C++ structs
- Data management with vectors
- Saving information to TXT files
- Console-based menu system

## Technologies Used

- C++
- Standard Library
- File I/O (`fstream`)
- Data Structures (`vector`)
